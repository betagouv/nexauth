# nexauth

> A dead simple JWT-based authentication for Next.js framework.

## Features

- Credential-based authentication
- Refresh Token Rotation
- Automatic Reuse Detection
- Strong EdDSA signature using SHA-3 and Curve448
- Prisma Adapter
